# 📦 Inventra - Inventory Management System

Inventra is a web-based Inventory Management System developed using **PHP** and **MySQL**. It enables businesses to efficiently manage products, suppliers, customer orders, inventory levels, and sales reports through a simple and user-friendly interface.

---

## 🌟 Features

### 📊 Dashboard
- Inventory overview
- Business summary
- Quick access to all modules

### 📦 Product Management
- Add new products
- Update product details
- Delete products
- View all products

### 🚚 Supplier Management
- Add suppliers
- Edit supplier information
- Delete suppliers
- Manage supplier database

### 🛒 Order Management
- Add customer orders
- View all orders
- Track inventory updates

### 📈 Inventory Management
- Monitor stock levels
- Inventory turnover analysis
- Least sold products
- Most sold products

### 📑 Reports
- Sales Report
- Product Report
- Inventory Turnover Report
- Business Analytics

### 🔐 Authentication
- User Registration

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/anushkaa-2006/Inventra-Inventory-Management-System.git
```

### 2. Move the Project

Copy the project folder into:

```
C:\xampp\htdocs\
```

### 3. Start XAMPP

Start the following services:

- Apache
- MySQL

### 4. Import the Database

- Open **phpMyAdmin**
- Create a database
- Import **invetra_admin.sql**

### 5. Configure Database

Update the database credentials in `db.php` if required.

### 6. Run the Project

Open your browser and visit:

```
http://localhost/inventory
```
